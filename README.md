# Tech Stack
`Frontend`
React (Vite)

TailwindCSS

React Router

React Markdown

`Backend`
Strapi CMS (v4)

SQLite (default) or any preferred database

REST API with auto-generated endpoints

# Features
`Frontend`
Fetch and display blog posts dynamically

Markdown rendering with headings, bold text, and inline images

Responsive UI with TailwindCSS

Blog detail page with cover image + content images

SEO-friendly structure

`Backend`
Easy blog creation through admin panel

Upload media (images)

Auto API generation

Relations support (cover image + rich text content)

Role & Permissions management
